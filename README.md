# Travel Site
Developed during Brad Schiff's Git a Web Job: Mastering the Modern Workflow Udemy course.

Travel site is a static portfolio website for a fictional travel agency.

Live website can be viewed on: https://christriane.github.io/travel-site/

## Getting Started

```
git clone https://github.com/christriane/travel-site.git
cd into the directory
gulp watch
```

## Gulp Build Production Code
```
gulp build
```

Gulp will generate a folder named docs in the app folder. This is the folder currently hosted on github pages.


## Tech/conventions used:
* Gulp
* Webpack
* PostCSS
* BEM
